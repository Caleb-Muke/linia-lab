# LINIA LAB — Site prototype

Prototype d'un site institutionnel pour LINIA LAB (Université de Kinshasa). Fichiers fournis:

- `index.html` — page unique (sections: Accueil, Projets, Actualités, Équipe, À propos, Contact)
- `styles.css` — styles principaux, responsive
- `scripts.js` — comportement (menu mobile, animations, compteurs)
- `assets/logo.svg` — logo SVG placeholder

Ouvrir `index.html` dans un navigateur pour voir le rendu local.

Prochaines étapes suggérées:
- Remplacer les images placeholder par vos propres visuels
- Intégrer un backend (PHP/Laravel ou Django) pour la gestion des projets/actualités
- Ajouter SEO, métadonnées et optimisation d'accessibilité approfondie
